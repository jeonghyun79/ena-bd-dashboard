# E&A BD Dashboard

GitHub Pages용 정적 대시보드입니다.

## 게시 방법
1. 이 폴더의 `index.html`을 GitHub 저장소 루트에 업로드합니다.
2. Settings > Pages > Build and deployment 에서 Source를 `Deploy from a branch`로 설정합니다.
3. Branch를 `main` / `/root`로 선택하고 Save 합니다.
4. 몇 분 뒤 `https://jeonghyun79.github.io/ena-bd-dashboard/` 주소로 접속합니다.
